# exsudats_racinaires
## Définition 
Les exsudats racinaires sont des composés chimiques libérés par les racines dans le sol.

## Types de molécules 
### Sucres 
Glucose, fructose, saccharose.

### Acides aminés 
Glutamate, aspartate, 

### Acides organiques 
malate, citrate

### Phénols, flavonoïdes, coumarines

### Composés signalétiques
